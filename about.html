About the project

A playful e-commerce concept for planets

Planet Shop turns the solar system into a premium storefront. The project is designed like a themed online store, with a landing page, a catalog, dedicated product pages, and a working cart.

Multi-page structure

The project now behaves more like a real front-end build instead of one long demo page.

Shared JavaScript

Catalog data, search, filters, product detail rendering, and cart logic are shared across the whole project.

Reusable styling

All pages use one shared design system so the storefront feels consistent and polished.
